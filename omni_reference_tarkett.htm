Спортивный линолеум Tarkett Omnisports Reference

(Таркетт Омниспорт Референс)

В 2017 году коллекция переименована в Tarkett Omnisports R65.

Tarkett Omnisports Reference — это спортивный линолеум из ПВХ для крытых спортивных площадок. Это напольное покрытие имеет высокую устойчивость к истиранию.

Спортивный линолеум Таркетт Омниспорт Референс обеспечивает амортизацию более 26%, что создаст комфортные условия как для занятий в тренажерном зале, так и для активных видов спорта.

Разнообразная расцветка спортивного линолеума Tarkett Omnisports Reference делает эту коллекцию идеальной для использования в спортивных залах образовательных учреждений, помещениях, которые предназначены для занятий гимнастикой или общей физической подготовкой.

Спортивное покрытие Tarkett Omnisports Reference станет достойной альтернативой паркету, благодаря техническим показателям и общей толщине материала: 6,5 мм.
