Опорные конструкции для грязезащитных решеток

Опорные конструкции для грязезащитных решеток – это сложные конструкции, которые используются для первой проходной зоны, в местах, где предусмотрено использование приямок. Опорные конструкции являются хорошим и прочным основанием для систем грязезащиты, это позволяет не только собирать различного рода загрязнения, но и создать систему водоотвода в ливневую канализацию.

Величина опорных конструкций для решеток может быть различной (от 40 мм и более), что позволяет подобрать наиболее подходящую конструкцию для Вашей территории, которая будет отвечать всем предъявляемым требованиям.

Для максимальной эффективности, опорная конструкция без труда регулируется по высоте, что позволяет ее точно подстроить под глубину приямка независимо от его глубины (если глубина приямка составляет 80 мм, то регулировать высоту опорной конструкции для грязезащитных решеток можно с помощью винтовых регулируемых опор).
