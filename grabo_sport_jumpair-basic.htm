Паркет Grabo JumpAir Basic для фитнес-зала

Паркет Grabo JumpAir Basic – это спортивное покрытие для фитнес-центров и танцевальных залов. Спортивный паркет Грабо используют также в тренировочных залах, студиях для гимнастики и йоги,в зонах кардиотренировок.

Паркет Grabo JumpAir Basic с верхним слоем из твердой породы дерева 3,5 мм подойдет и для детских центров, игровых зон. Гидроизолирующая прокладка защищает напольное покрытие от попадания влаги с основания.

Паркет Грабо JumpAir Basic для фитнес-зала имеет высокие показатели по отскоку мяча и скольжению, поэтому этот материал подойдет для проведения игровых видов спорта.
