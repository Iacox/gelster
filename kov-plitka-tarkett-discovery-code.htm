Ковровая плитка в полоску Tarkett Discovery Code

Ковровая плитка в полоску Discovery Code – новинка 2018 года от фабрики Tarkett. Это покрытие с петлевым ворсом выпускается в трех цветовых решениях. Покрытие входит в группу коллекций Discovery, вдохновленных путешествиями в самые отдаленные уголки земли.


Полосатая ковровая плитка Tarkett Discovery Code помогает снизить уровень ударного шума в помещении. Это позволяет использовать ее в людных офисах, конференц-залах, кинотеатрах, ресторанах и кафе. Плитка также не дает скапливаться статическому электричеству, поэтому подойдет для опенспейсов с большим количеством компьютеров.
